# Digital Flipbook - Interactive PDF Presentation Viewer

An interactive web application for viewing PDF presentations with multiple viewing modes.

## Features

✨ **Viewing Modes**
- 📖 **Flipbook View** - Classic twin-page book experience
- 🎯 **Grid View** - Browse all pages as a thumbnail gallery
- 📱 Full responsive design

⚡ **Interactive Features**
- ⌨️ Keyboard Navigation (Arrow keys, Spacebar)
- 🎬 Smooth animations and transitions
- 📊 Progress tracking
- 🔍 Jump to specific pages
- 🖥️ Fullscreen mode

## Quick Access

1. **Original Flipbook**: https://marvinvgamban-ai.github.io/digital-flipbook/
2. **Cartesian Viewer (Grid)**: https://marvinvgamban-ai.github.io/digital-flipbook/cartesian-viewer.html

## Troubleshooting

**PDF not loading?**
- Clear browser cache (Ctrl+Shift+R)
- Check GitHub Pages deployment in Settings → Pages
- Verify PDF filename matches exactly
- Check browser console for errors

**Not working at all?**
- Ensure repository is public
- Verify GitHub Pages is enabled:
  - Settings → Pages
  - Source: `Deploy from a branch`
  - Branch: `main`
  - Folder: `/ (root)`
